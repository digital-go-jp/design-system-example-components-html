# デジタル庁デザインシステム コードスニペット（HTML版）

[デジタル庁デザインシステム](https://design.digital.go.jp/)のコンポーネントの一部をプレーンなHTML、CSS、JavaScriptで実装したサンプル集です。

[React版](https://digital-go-jp.github.io/design-system-example-components/)もご利用いただけます。

## HTML版の特徴

<dl>
<dt>アクセシビリティファースト</dt>
<dd>
  WCAG等のアクセシビリティガイドラインを最大限に取り込んだデジタル庁デザインシステムのガイドラインを基に作られており、アクセシビリティの専門家によるチェックも受けています。
</dd>
<dt>フレームワーク非依存</dt>
<dd>
  ReactやVue.jsなど、特定のフレームワークに依存せず、どんな技術スタックでも利用可能です。
</dd>
<dt>カスタマイズ可能</dt>
<dd>
  HTML、CSS、JavaScriptすべてのコードが改変可能で、プロジェクトのニーズに合わせて調整できます。
</dd>
<dt>後付け可能</dt>
<dd>
  リセットCSSの有無を問わず動作し、既存のウェブサイトに後から組み込むことができます。
</dd>
<dt>シンプルな実装</dt>
<dd>
  複雑な抽象化や高度なCSS機能を避け、シンプルで理解しやすいコードで構成されています。
</dd>
</dl>

## リソース

デジタル庁が提供する、デジタル庁デザインシステムのガイドラインやデザインデータなど、連携可能な各種リソースです。用途の必要性に応じてあわせてご利用ください。

- [デジタル庁デザインシステムウェブサイト（ガイドライン）](https://design.digital.go.jp/)
- [デザインデータ（Figma）](https://www.figma.com/community/file/1377880368787735577)
- [ウェブアクセシビリティ導入ガイドブック](https://www.digital.go.jp/resources/introduction-to-web-accessibility-guidebook)
